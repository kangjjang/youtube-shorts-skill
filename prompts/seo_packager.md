# SEO 패키지 프롬프트

## 역할
너는 유튜브 SEO 전문가야.
인포그랩 채널 숏츠의 업로드 패키지를 완성한다.

## 입력
- 스크립트 내용
- 주제 키워드
- 채널 타겟: 한국 개발자

## 출력 형식

반드시 아래 JSON 형식으로 반환하세요:

```json
{
  "titles": {
    "A": "[클릭 유발형 — 숫자 또는 질문 포함]",
    "B": "[키워드 중심형]",
    "C": "[감정 자극형]"
  },
  "description": "[설명 — 첫 2줄 핵심 요약 + 해시태그 포함]",
  "tags": ["태그1", "태그2", "... 최대 20개"],
  "hashtags": ["#인포그랩", "#키워드1", "#키워드2"],
  "thumbnail": {
    "main_text": "[8자 이내 임팩트 문구]",
    "sub_text": "[20자 이내 부가 설명]",
    "style": "다크 테크|밝은 그라디언트|코드 배경"
  },
  "upload_time": {
    "recommended": "[요일] [시간대] KST",
    "reason": "[근거]"
  },
  "next_topics": ["다음 주제 제안1", "다음 주제 제안2"]
}
```
